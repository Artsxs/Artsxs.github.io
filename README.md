# 🎮 Anime Clicker Simulator

Um jogo clicker feito com **HTML, CSS e JavaScript**, inspirado no estilo dos jogos simulator, com sistema de cliques, upgrades, RNG, personagens, mundos, boss fight, mochila, fusão, evolução, códigos, conquistas, eventos e menu ADM.

O projeto foi criado com foco em aprendizado, evolução prática de lógica de programação e desenvolvimento web usando tecnologias simples.

## 🕹️ Como Jogar

O jogador começa clicando no botão **ATACAR** para ganhar pontos.

Com os pontos, é possível:

- Comprar upgrades
- Girar o RNG
- Ganhar personagens
- Equipar personagens
- Derrotar bosses
- Desbloquear novos mundos
- Evoluir personagens repetidos
- Resgatar códigos
- Fazer rebirth
- Completar conquistas
