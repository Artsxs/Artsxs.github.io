# 🎮 Anime Clicker Simulator

**Anime Clicker Simulator** é um jogo estilo **Clicker / Simulator / RNG** feito com **HTML, CSS e JavaScript puro**.

O objetivo do jogo é clicar para ganhar pontos, comprar upgrades, girar RNG para conseguir personagens, equipar os melhores, evoluir personagens repetidos, desbloquear mundos, derrotar bosses, usar pets, auras, tickets, rebirth e evoluir cada vez mais.

Este projeto foi criado com foco em aprendizado de programação web, lógica de jogo, interface, sistemas de progressão e organização de código.
